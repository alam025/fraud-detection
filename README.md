## 🚀 About

> **Achieved 99.1% AUC-ROC with <5ms per-transaction inference latency on Kafka stream**

**Fraud Detection System** is a production-ready **Data Science** project built with **Python, LightGBM, Kafka, Grafana**.
Deployed on **AWS EC2** with full CI/CD pipeline.
